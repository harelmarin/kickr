<a name="english"></a>
## English

### Description
Kickr is a web application designed for football enthusiasts. It allows users to follow competitions, check upcoming matches, and access detailed information about sports events in real-time.

The project is currently deployed and accessible online.

### Tech Stack

#### Backend
The server is built with Java and the Spring Boot ecosystem.
- **Framework**: Spring Boot 3
- **Database**: PostgreSQL
- **Security**: Spring Security with JWT authentication
- **Documentation**: Swagger UI / OpenAPI
- **Services**: Cloudinary (image storage), Resend (email delivery)

#### Frontend
The user interface is developed with React and TypeScript.
- **Build Tool**: Vite
- **Styling**: TailwindCSS
- **Data Management**: React Query and Zustand
- **Navigation**: React Router

### Project Status
The application is currently in production.
